  Creating a Script

Creating a script is done by clicking Script > New. This will pop-up the Create Script Dialog. The dialog takes a number of parameters and creates a new script based on the current stage scene.

Parameters

  1. Minimum Time is the beginning time of the script.
  2. Maximum Time is the ending time of the script.
  3. Unit of Time is a string specifying the time unit.
  4. # of Scenes is the number of scenes in the script.
  5. Scene Rate is the number of scenes per second displayed during animation.
  6. Interpolate is the interpolation method for the camera position.
When Ok is clicked, a script will be create with the user specified number of scenes. All scenes will have the same actors, lights, and cameras as the current stage scene. The time of each scene will be at equal intervals starting at the minimum time and finishing at the maximum time. The script will be loaded into the viewer and the first scene will be displayed on stage.
