  Actors

Actors consist of three parts; the graphic representation (the colored grid), the coordinate representation (the axis), and the property representation (the color bar).

The actors are displayed under the Actors tab on the left hand side of the viewer. Right clicking a highlighted actor or set of highlighted actors will open a menu to manipulate the them.
  • New [Ctrl-N] will add an actor to the stage using SheafScope. See the SheafScope Usage documentation on how to create an actor.
  • Edit Actor(s) [Ctrl-E] will open a dialog for editing the highlighted actor(s). The dialog has a number of tabbed panels for manipulating the actor. After making the desired changes, Apply will apply the changes. Cancel will restore the settings back to the last applied values.
  • Remove Actor(s) [Ctrl-R] will remove the highlighted actor(s) from the stage.
  • Select Actor [Ctrl-S] will select the highlight actor. The selected actor is identified by bold red text. The coordinate and property representations of the selected actor are displayed on the stage.
  • View Dofs [Ctrl-V] will display the coordinate and properties dofs for the highlighted actor.

The Actor menu on the top menu bar is a shortcut for the selected actor. Actor > Reset Actor will reset the actor to the values of the selected scene in the script.
