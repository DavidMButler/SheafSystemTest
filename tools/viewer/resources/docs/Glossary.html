Glossary


Actor – A VTK representation of a coordinate section and any number of property sections.

Camera – A VTK camera object.

Current Stage Scene – The Scene currently being displayed on the Stage.

Highlighted – The result of a single left mouse click on a list. The background color of the list item will change to a light blue.

Light – A VTK light object.

Multi-Scene Script – A Script with more than one Scene.

Quicktime® – A movie file format. Quicktime® is a registered trademark of Apple Computer corporation.

Scene – A collection of Actors, Lights, and Cameras at a given time.

Script – A collection of sequential Scenes.

Selected – The list item with bold red text.

Single-Scene Script – A Script with only one Scene.

Stage – The 3D space a Scene is being acted out.

Viewport – The view of the Stage through a Camera.

VTK – Visualization Toolkit by Kitware.
