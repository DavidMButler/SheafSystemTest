  Lights

The lights are displayed under the Lights tab on the left hand side of the viewer. Right clicking a highlighted light or set of highlighted lights will open a menu to manipulate them.
  • New [Ctrl-N] will add a light to the stage. The initial position and focal point of the new light will be the same as the selected camera but will not follow the camera.
  • Edit Light(s) [Ctrl-E] will open a dialog for editing the highlighted light(s). The dialog has a number of tabbed panels for manipulating the light. After making the desired changes, Apply will apply the changes. Cancel will restore the settings back to the last applied values.
  • Remove Light(s) [Ctrl-R] will remove the highlighted light(s) from the stage.
  • Select Light [Ctrl-S] will select the highlight light. The selected light is identified by bold red text.

The Light menu on the top menu bar is a shortcut for the selected light. Light > Reset Light will reset the light to the values of the selected scene in the script.
