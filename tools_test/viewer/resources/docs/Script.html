  Script

A script is a collection of sequential scenes. Scripts can be created, saved and opened through the Script menu on to top menu bar. Scripts are used by the viewer to create key frame animations and record movies. The LPS Viewer always has exactly one script loaded.

Script > Reset Scene will reset the current stage scene to the selected scene in the script. The selected scene in the script is the scene in bold red text under the Script tab on the left hand side of the viewer.

The scenes of a script are displayed under the Script tab on the left hand side of the viewer. Double clicking a scene will load it onto the stage.
