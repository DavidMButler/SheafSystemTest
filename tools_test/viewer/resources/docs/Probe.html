  Probe

To display the Probe Dialog, click Edit > Probe. This dialog allows points to be selected by first selecting the enable button, then moving the cursor over the actor to be probed and selecting the 'p' key. The results of the probe appear in the text area. If a point which is not part of an actor is selected, the message "Nothng picked" will be displayed.
