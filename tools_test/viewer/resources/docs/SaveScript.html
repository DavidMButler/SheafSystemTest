  Saving/Opening a Script

A script can be saved to a CSV file. Script > Save > Scene will save the current stage scene as a single-scene script. Script > Save > Script will save the script currently loaded into the LPS Viewer.

Saved files can be opened directly from the SheafScope (see the SheafScope User Guide). It can also be opened from the viewer by clicking Script > Open. Opening a script will cause it to be loaded into the viewer and the first scene of the script displayed on the stage.

Note: Currently scripts are associated with a Sheaf Namespace. The namespace must be loaded in SheafScope in order for the script to be successfully loaded.
