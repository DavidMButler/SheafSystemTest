  Editing a Script

A script can be edited using key frame animation.
  1. The script must be in edit mode. Edit mode can be entered by clicking Script > Edit Mode or by clicking the edit script mode toggle button on the left hand side of the toolbar.
  2. Once in edit mode, scenes can be marked as key frames. Under the Script tab on the left hand side of the viewer is a list of scenes. A scene can be marked as a key frame by double clicking it or by highlighting the scene, right clicking the mouse, then clicking Edit Key Frame. A key icon will appear beside the scene when it is marked as a key frame.
  3. Once the scene is marked as a key frame and is loaded onto the stage (i.e., it is displayed in bold red text), it is ready to be edited.
  4. Edit as many scenes as necessary (Steps 2-3).
  5. After all key frames have been edited, the script is ready to be interpolated. Right click the mouse over the Script tab then click Interpolate to interpolate the script.
Note: All numerical, color, and other constant values are boolean interpolated. The camera position is linearly interpolated in polar coordinates. All other vector values (such as actor position, actor orientation, actor scale, ...) are linearly interpolated without polar transformation.
