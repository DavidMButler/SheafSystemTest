  SheafScope Usage

SheafScope is a browser for Sheaf Namespaces. SheafScope provides an interface for visualizing sections within a Sheaf Namespace.

Contents:

  1. SheafScope is based on the Java Swing components and uses the pluggable look and feel capability of Swing. If the Theme menu is activated, the user can change the look and feel of SheafScope interactively.
