  Visualizing Sections

SheafScope provides two interfaces for visualizing sections using the LPS Viewer:
  • Creating a View
    1. A view can be created by clicking Visualize > Create View.
    2. The Create View Dialog will be opened, allowing the user to choose the type of view to be created. The list of supported views are based on graphic representations.
    3. Once a view is selected, a toolbar will appear at the top of SheafScope. The user can fill in the sections of the view by clicking the appropriate radio button then selecting the section using the namespace browser.
    4. After the toolbar is filled in, View will create the graphic representation.
    5. The Viewer Dialog will be opened, allowing the user to choose to visualize the representation in a new LPS Viewer or an existing one.
    6. Reset will clear the sections from the toolbar. Cancel will remove the toolbar from SheafScope.

  • Opening a Script – The LPS Viewer allows the user to save a script file (see LPS Viewer documentation). A script file can be opened from SheafScope by clicking Visualize > Open Script. It is important to note, scripts are currently associated with a Sheaf Namespace and the namespace must be loaded in order for a script to be opened. Scripts will be opened in a new LPS Viewer.
Script > Open Viewer allows the user to open an existing LPS Viewer.
